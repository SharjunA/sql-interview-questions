# SQL Interview Questions 🚀

A curated collection of SQL interview questions and solutions designed to help you crack technical interviews in Data Engineering, Data Analysis, and Software roles.

---

## 📌 About This Repository

This repository contains a structured set of SQL problems ranging from basic to advanced difficulty. It is designed to:

* Strengthen SQL fundamentals
* Improve problem-solving skills
* Prepare for real-world interview scenarios

---

## 🧠 Topics Covered

* Basic SQL Queries (SELECT, WHERE, GROUP BY)
* Joins (INNER, LEFT, RIGHT, SELF)
* Subqueries & Correlated Queries
* Window Functions (ROW_NUMBER, RANK, LAG, LEAD)
* Aggregations & Analytics
* Case-based Interview Questions

## 💡 Example Question

**Find the second highest salary**

```sql
SELECT DISTINCT salary
FROM employee
ORDER BY salary DESC
LIMIT 1 OFFSET 1;
```
